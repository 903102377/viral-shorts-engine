# 🚀 零成本防封黑科技：Codespace + WARP 部署全流程指南

> [!TIP]
> 恭喜您成功白嫖了这条千万级电商工作室才用得上的“纯净高速隧道”！
> 这份指南将告诉您，在每次电脑关机、或者 Codespace 休眠后，**下次如何在一分钟内光速满血复活！**

---

### 第一步：在云电脑（Codespace）中唤醒 WARP 幽灵
由于您的 Codespace 是通过 `gh cs` 开启的独立虚拟容器，每次它一旦沉睡再醒来，底层常驻的 WARP 后台进程就会“装死”。您需要把它重新拉起。

1. 用任意方式（或者直接在网页端）进入您的 Codespace 终端黑框框。
2. 复制并回车下面这句“亡灵召唤术”（哪怕代码空间重启，这句能让它强制隐态复活）：
   ```bash
   nohup sudo warp-svc --accept-tos > /dev/null 2>&1 &
   ```
3. *(选做边界检查)* 如果您不放心它有没有连上，可以补一句：
   ```bash
   warp-cli --accept-tos connect
   ```
   只要云端代理绑在我们设定好的 `40000` 端口上，第一步就算完成！

---

### 第二步：在您的 Mac 本机打通神级隧道
千万别忘了，云电脑里跑起来没用，我们得把这根线“接”导回 Mac 上。

1. 在您的 Mac 本机，打开终端（就是您之前本地输代码的地方）。
2. 输入下面这行神仙端口直连代码（注意替换成您当前活跃的 Codespace ID）：
   ```bash
   gh cs ssh -c cautious-space-adventure-qwjj6x6w7442x66 -- -L 1081:127.0.0.1:40000
   ```
   > **⚠️ 核心要点：**
   > *   这串代码敲下去，只要登录进去了，**这个黑框框就绝对不能关**！关了桥就塌了！
   > *   如果您以后新开的云空间名字变了（比如叫 `happy-tree...`），记得把 `-c` 后面的名字换成最新的！

---

### 第三步：切通项目的血脉
隧道搭好后，您本机的 `127.0.0.1:1081` 就不再是一个普通的数字，它等同于“身披 Cloudflare 纯净铠甲的隐形战机”。

1. 您平常看网页用的**浏览器插件（例如 SwitchyOmega）**：把刚才建好的 SOCKS5（127.0.0.1 端口 1081）点亮即可。
2. **跑自动化项目**：不需要额外做任何改变（如果是手动操控）。如果您有额外的环境变量或 `flow-automator` 需要走代理，此时让 Playwright 启动时识别本机系统的 `1081` 代理环境即可（通常全局梯子即可涵盖）。
3. 直接执行 `npm run dev`，您会发现 Google 给您开了一路绿灯，100% 成功率暴力产出不再是梦！
